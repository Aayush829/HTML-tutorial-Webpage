HTML Block and Inline Elements

Every HTML element has a default display value, depending on what type of element it is. 

There are two display values:

1. Block-level Elements:

A block-level element always starts on a new line and block-level element always takes up the full width available (stretches out to the left and right as far as it can). A block level element has a top and a bottom margin, whereas an inline element does not.

The <div> element is a block-level element.

For Example:

<div>Hello World</div >

2. Inline Elements:

An inline element does not start on a new line.An inline element only takes up as much width as necessary.

The <span> element is a inline element.

For Example:

<span>Hello World</span>
