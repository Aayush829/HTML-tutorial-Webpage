HTML Lists

HTML lists allow web developers to group a set of related items in lists. 

Genrally there are Two types of Lists:

1. Ordered HTML List:

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

For Example:

<ol>

    <add 1 cup water and get to a boil.>

    <add the noodles to the pan and pour in the contents of the flavor packet.>

    <mix well and boil for 2 minutes>

    <enjoy masaledar maggi.>

</ol>
